# PRD: 성락교회 웹페이지

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 성락교회 웹페이지 리뉴얼 |
| **목표** | 노후화된 웹페이지를 세련되고 고급스럽게 리뉴얼하여 누구나 오고 싶은 교회 이미지 구축 |
| **개발 방식** | FE 우선 개발 (Mock API) → 추후 Spring Boot BE 연동 |

## 2. 타깃 사용자

### Primary: 기존 성도님들
- 예배 정보 확인
- 설교 영상 시청
- 교회 소식 확인

### Secondary: 교회를 찾는 새로운 분들
- 교회 소개 정보 탐색
- 예배 시간/장소 확인
- 새가족 등록 문의

## 3. MVP 핵심 기능

### 3.1 유튜브 연동 (설교 중심)
- YouTube API로 재생목록 기반 자동 수집
- 재생목록: 주일설교, 수요예배, 새벽기도
- 메인: 최신 주일설교 1개 크게 + 리스트 3개
- 설교 페이지: 전체 목록 (재생목록별 필터)
- 유튜브 라이브 임베드 (예배 실황)

### 3.2 교회 소개
- 담임목사 인사말
- 교회 비전
- 연혁
- 조직도
- 오시는 길

### 3.3 새가족 영접
- 새가족 안내 페이지
- 문의 폼 (이름, 연락처, 메시지)

## 4. 전체 기능 목록

### 4.1 공개 페이지

| 페이지 | 주요 기능 |
|--------|----------|
| **메인 (홈)** | 히어로 섹션, 라이브/최신 설교, 교회 소개 요약, 공지사항, 새가족 환영 |
| **교회소개** | 인사말, 비전, 연혁, 조직도, 오시는길 (지도) |
| **예배안내** | 예배 종류별 시간/장소 |
| **설교/영상** | 유튜브 재생목록별 영상 목록, 검색, 필터 |
| **새가족안내** | 안내 정보 + 문의 폼 |
| **교회소식** | 공지사항, 소식 목록 (카테고리별) |
| **문의/연락처** | 연락처 정보, 문의 폼 |

### 4.2 관리자 페이지

| 페이지 | 주요 기능 |
|--------|----------|
| **대시보드** | 최근 문의, 통계 요약 |
| **공지사항 관리** | CRUD, 카테고리 관리 |
| **예배정보 관리** | 예배 시간/장소 수정 |
| **교회소개 관리** | 각 섹션 콘텐츠 수정 |
| **라이브 설정** | 라이브 URL 설정, ON/OFF |
| **새가족 문의** | 문의 목록 조회, 처리 상태 관리 |
| **설교 동기화** | 유튜브 재생목록 수동 동기화 |
| **관리자 계정** | 계정 관리 (최고관리자만) |

## 5. 디자인 방향

| 항목 | 방향 |
|------|------|
| **톤앤매너** | 밝고 따뜻함 + 고급스러움 |
| **컬러** | 화이트 베이스 + 골드/베이지 포인트 |
| **애니메이션** | 화려하게 (패럴랙스, 인터랙티브 요소) |
| **메인 시안** | 다양한 시안 제공 예정 |

## 6. 기술 스택

### Frontend
- React (Vite 또는 Next.js)
- TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- TanStack Query + Zustand

### Mock 환경
- MSW (Mock Service Worker)
- LocalStorage
- Faker.js

### 추후 Backend
- Java Spring Boot
- PostgreSQL
- Redis

## 7. 비기능 요구사항

- **반응형**: 모바일, 태블릿, 데스크톱 대응
- **SEO**: 메타태그, 시맨틱 마크업
- **성능**: LCP 2.5초 이내
- **접근성**: WCAG 2.1 AA 수준
- **브라우저**: Chrome, Safari, Edge 최신 버전
