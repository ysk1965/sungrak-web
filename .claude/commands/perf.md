React/Next.js 성능 분석기입니다.

$ARGUMENTS 에 파일/폴더 경로가 있으면 해당 범위를 분석합니다.
없으면 프로젝트 전체를 스캔합니다.

## 분석 항목

### 1. 번들 사이즈
- [ ] 큰 라이브러리 import (lodash 전체 import 등)
- [ ] tree-shaking 안 되는 패턴
- [ ] 동적 import 가능한 곳 (next/dynamic, React.lazy)
- [ ] 이미지/폰트 최적화 상태

### 2. 렌더링 최적화
- [ ] 불필요한 re-render 유발 코드
  - 인라인 객체/함수 props
  - Context 분리 안 된 것
  - 상태 위치가 너무 높은 것
- [ ] memo, useMemo, useCallback 필요한 곳
- [ ] key prop 문제 (index 사용, 불안정한 key)

### 3. 데이터 페칭
- [ ] 워터폴 요청 (순차 호출)
- [ ] 캐싱 전략 (SWR, React Query 활용)
- [ ] 프리페칭 가능한 곳
- [ ] 서버 컴포넌트 활용 가능한 곳 (Next.js 13+)

### 4. Core Web Vitals 영향
- [ ] LCP: 큰 이미지, 느린 폰트 로딩
- [ ] FID/INP: 무거운 이벤트 핸들러, 메인 스레드 블로킹
- [ ] CLS: 레이아웃 시프트 유발 요소
